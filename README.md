**SCU 统一身份认证 自动登录 Userscript**

- **简介**: 这是一个用于四川大学统一身份认证页面的油猴/Userscript，自动使用外部 OCR 识别验证码并尝试自动填写学号与密码并提交登录。

- **文件**: 脚本位于 `scu-auto-login.user.js` （工作区根目录）。

- **安装**:
  - 安装 Tampermonkey/Violentmonkey（浏览器扩展）。
  - 打开扩展 -> 新建脚本，粘贴 `scu-auto-login.user.js` 的内容并保存，或直接用扩展的“安装本地脚本”功能。

- **配置**:
  - 在脚本顶部直接修改常量：
    - `OCR_PROVIDER`：你的 OCR 服务 POST 接口，接收 `{ img: base64 }` 并返回 JSON（建议返回字段 `result` 或 `text`）。
    - `STUDENT_ID`：学号（用于自动填写用户名）。
    - `PASSWORD`：登录密码（注意安全，谨慎保存）。

- **行为**:
  - 页面加载或验证码图片刷新时自动触发 OCR 并填写验证码。
  - 会先填充学号，再填充识别得到的验证码，最后填写密码并尝试提交。
  - 重试限制：1 分钟窗口内最多尝试 5 次，失败时会尝试点击验证码图片或可能的刷新按钮并在 5 秒后重试一次。

- **隐私与安全**:
  - 脚本会把密码直接写在脚本文件中（明文），请仅在受信任的环境中使用。
  - OCR 服务会接收验证码图片的 base64，可能会上传到第三方服务器，请确认 OCR 服务的隐私策略。
